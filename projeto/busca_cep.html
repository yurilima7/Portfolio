Descrição do projeto

O Busca Cep é um aplicativo que facilita a consulta de informações de locais por meio de CEPs e a busca de CEPs com base em endereços fornecidos. Usando a API ViaCEP, o app oferece uma maneira rápida de obter dados detalhados sobre locais e endereços. Este projeto foi desenvolvido com o objetivo de praticar conceitos como gerenciamento de estado com MobX e a implementação de testes unitários, fortalecendo a compreensão dessas práticas no desenvolvimento de aplicações móveis.

Desenvolvimento do projeto

O desenvolvimento do Busca Cep começou com a escolha da API ViaCEP para fornecer as informações necessárias para a busca de endereços e CEPs. Após a definição das funcionalidades principais, as telas do app foram prototipadas no Figma, garantindo uma navegação simples e intuitiva. A aplicação foi desenvolvida em Flutter, integrando pacotes para consumo da API e gerenciando o estado com MobX. O projeto foi concluído em 2 semanas, com atenção ao tratamento de erros identificados durante o processo de testes.
