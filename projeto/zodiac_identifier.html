Descrição do projeto

O Zodiac Identifier é um aplicativo que identifica constelações do zodíaco a partir de imagens selecionadas da galeria do smartphone. Este projeto foi desenvolvido como parte da disciplina de Processamento de Imagens no curso de Ciência da Computação, aplicando técnicas de visão computacional para reconhecimento de padrões celestes.

Desenvolvimento do projeto

O desenvolvimento do Zodiac Identifier começou com a busca de uma base de dados de imagens de constelações para o treinamento do modelo. A base disponível continha menos de 800 imagens, o que me levou a realizar manualmente as marcações e descrições das constelações em cada imagem. Utilizei o Google Colab para o treinamento do modelo com a YoloV5, um framework de detecção de objetos eficiente para esse tipo de aplicação.

Com o modelo treinado, criei uma API simples usando FastAPI, que processa as imagens enviadas pelo aplicativo. O app, desenvolvido em Flutter, conta com uma interface onde o usuário pode selecionar uma imagem da galeria. A imagem é então enviada para a API, que utiliza o modelo para identificar a constelação e retornar o nome, junto com a porcentagem de confiança na classificação.
