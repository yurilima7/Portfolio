Ticket IF - Aplicativo de Solicitação de Refeições IFMA

Descrição do projeto

O TicketIF é um aplicativo desenvolvido para o IFMA Campus Caxias com o objetivo de simplificar o processo de solicitação de refeições no campus. Os alunos podem solicitar almoço e jantar em duas modalidades: diária, para uso no mesmo dia, ou permanente, com validade até o fim do semestre no dia escolhido. Além da funcionalidade voltada para os alunos, o TicketIF inclui módulos específicos para o Restaurante e a Coordenadoria de Assuntos Estudantis (CAE).

A CAE é responsável pela aprovação das solicitações e pode gerar relatórios diários ou mensais das refeições, exportáveis em formato XLS, além de gerenciar turmas e realizar a limpeza da base de dados no final de cada semestre. O Restaurante tem acesso a relatórios diários para planejar a produção das refeições, além de utilizar QR Codes para confirmar o consumo por parte dos alunos.

Desenvolvimento do projeto

O desenvolvimento do TicketIF começou com a modelagem de um banco de dados relacional para gerenciar as informações de alunos, refeições e autorizações. Utilizamos o FastAPI para criar a API que integra os diferentes módulos do app. O processo de desenvolvimento foi dividido em etapas: primeiro, focamos no módulo do estudante e suas principais funcionalidades, como solicitação de refeições e visualização do status dos pedidos.

Em seguida, desenvolvemos o módulo da CAE, com ferramentas para a aprovação de pedidos e a geração de relatórios. Por fim, implementamos o módulo do restaurante, incluindo o acesso aos relatórios diários e o sistema de verificação via QR Code. O projeto foi desenvolvido ao longo de 1 ano e seis meses, como parte do estágio no curso de Ciência da Computação.
