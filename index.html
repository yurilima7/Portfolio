SOBRE MIM

Tenho 22 anos e atualmente sou acadêmico de Ciência da Computação no Instituto Federal do Maranhão Campus Caxias, além de ser desenvolvedor mobile Flutter. Apaixonado por tecnologia, adoro estar sempre atualizado com as últimas tendências do mercado.

Como desenvolvedor mobile, tenho experiência em design de interfaces de usuário (UI) e experiência do usuário (UX), bem como em desenvolvimento de aplicativos para Android e iOS utilizando o framework Flutter. Além disso, tenho habilidades em linguagens de programação como Dart, Java e Python e também um bom domínio de bancos de dados como MySQL e Firebase para integração de dados em tempo real.

Na área de desenvolvimento web, estou empenhado em aprimorar minhas habilidades em linguagens como HTML, CSS e JavaScript, bem como em React e Next JS. Meu entusiasmo por empreendedorismo me impulsiona a expandir meu conhecimento técnico e a aplicar minha habilidades de programação em projetos inovadores.
